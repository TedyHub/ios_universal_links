# iOS Universal Links

This project is an example of how to use Universal Links in an iOS application. Universal Links allow you to link to content inside your app from your website. When a user clicks on a Universal Link, the operating system will automatically open your app and navigate to the correct content.

## Features

* A simple iOS app with a single view.
* Configuration for Universal Links.
* An example of how to handle Universal Links in the app.

## Requirements

* Xcode 13 or later
* iOS 15 or later
* A registered domain name
* An Apple Developer account

## Installation

1. Clone this repository to your local machine.
2. Open the project in Xcode.
3. Change the bundle identifier to a unique value.
4. Change the team to your own team.
5. Configure your domain name in the project.
6. Create an `apple-app-site-association` file and upload it to your website.

## Usage

1. Build and run the app on a physical device.
2. Open a link to your website in Safari.
3. The app should open automatically.
